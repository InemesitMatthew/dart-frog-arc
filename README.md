# Dart Frog Arc

Come along while I grind my way toward **fluency in Dart on the backend**. Not just “it runs,” but architecture, APIs, and the habits that make servers boring in the best way.

This repo is my scratchpad and proof of work: small [Dart Frog](https://dartfrog.vgv.dev/) projects, CRUD experiments, WebSockets, and whatever lesson I’m chewing on next. If you’re learning the same stack, maybe something here saves you a detour.

## What’s inside

| Path | Roughly |
|------|--------|
| `server_01/` | Early server routes and basics |
| `web_socket_counter/` | WebSocket + counter example |
| `Learning CRUD/` | Todo APIs: simple CRUD, cleaner layering, multi-package layout |

Each folder is its own little app or exercise; jump in where the topic matches what you want to practice.

## Why Dart Frog?

Single-language backend in Dart, file-based routes, middleware you can reason about. Good fit if you already live in Flutter/Dart and want the same tooling and types end to end.
